> **Foodar** transforms the traditional restaurant experience into a futuristic, interactive journey. Designed for premium dine-in restaurants, it allows customers to scan a QR code at their table, visualize food in **Augmented Reality (AR)**, and order seamlessly.

---

## ✨ Key Features

### 🍽️ Smart Dine-In Flow
Forget traditional menus. Scan the QR at your table and get instant access to the digital kitchen.
- **Table-Specific Ordering**: Pre-configured for table service (e.g., Table No. 4).
- **Service Timing**: Choose between **"Serve Now"** for instant cravings or **"Schedule for Later"**.
- **Real-time Kitchen Tracking**: Watch your order progress from "Chef is Cooking" to "Ready to Serve".

### 🥽 Augmented Reality (AR) Visualization
See it before you eat it. 
- Use built-in AR views to visualize 3D models of dishes right on your table.
- High-fidelity textures and realistic lighting ensure a "what you see is what you get" experience.

### 🤖 AI-Powered Food Builder
Not sure what to eat? Let our AI Chef help.
- **Custom Builder**: Describe your mood or cravings, and the AI will craft a personalized meal for you.
- **AI Suggestions**: Smart pairing suggestions in the cart (e.g., "This Cold Coffee pairs perfectly with your Burger!").

### 👥 Group Ordering & Smart Split
Dining with friends? We’ve made the bill simple.
- **Live Group Mode**: Invite friends to add items to a shared cart.
- **Bill Calculator**: Automatically splits the subtotal and service charges among group members.

### 💎 Premium "Bubble Glow" Aesthetic
A high-end UI designed to "WOW" your customers.
- **Glassmorphism**: Elegant translucent panels and backdrop blurs.
- **Micro-animations**: Powered by Framer Motion for a fluid, living interface.
- **Dark Mode Optimized**: A stunning, high-contrast dark theme for evening dining.

---

## 🛠️ Tech Stack

- **Core**: React 18 + TypeScript
- **Routing**: TanStack Router (Type-safe routing)
- **Styling**: Tailwind CSS + Custom Vanilla CSS (Design Tokens)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mishraji018/fooder-order.git
   cd fooder-order
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Visit the app**:
   Open [http://localhost:8080](http://localhost:8080) in your browser.
